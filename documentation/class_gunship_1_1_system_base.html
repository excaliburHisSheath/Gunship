Detailed Description

The base class for all Gunship systems.

SystemBase defines the type Family and the variable familyCounter, which are used to give each System a unique identity. This is currently not put to use within Gunship, but is maintained for the purposed of compatibility with other EntityX systems, and may be utilized in the future.
