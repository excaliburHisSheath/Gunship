Detailed Description

Provides a resizable, semi-contiguous pool of memory for constructing objects in. Pointers into the pool will be invalided only when the pool is destroyed.

The semi-contiguous nature aims to provide cache-friendly iteration.

Lookups are O(1). Appends are amortized O(1).
