Detailed Description

Component for representing the position, rotation, and scale of an object.

The Transform component contains the local position, rotation, and scale, and allows retrieval of the transform in world-space.

Remarks
This component wraps a reference to an Ogre::SceneNode since we let Ogre manage the scene graph for us at this point. This may change as things go forward.
